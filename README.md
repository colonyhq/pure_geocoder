# PureGeocoder

A Geocoder that will work by scraping OpenStreetMap. 

## Getting Started
+ `$ git clone https://github.com/jackHedaya/pure_geocoder`
+ `$ cd pure_geocoder`
+ `$ mix deps.get`

## Installation
Currently in development. Installation will be available from https://hex.pm after release.
